Here you can find an overview of all possible checks. It's also possible to search for them using the search bar by pressing `ctrl + K`.

## Node Checks

- [Description checks](/docs/checks/description)
  - `has_description`: Check if a description is populated.

- [Naming Checks](/docs/checks/naming_convention)
  - `name_convention`: Check if objects name follows the given pattern

- [Tag Checks](/docs/checks/tags)
  - `has_tags`: Check if objects have the required tags.
